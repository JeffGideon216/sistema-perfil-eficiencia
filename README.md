# sistema-perfil-eficiencia
 Sistema Perfil de Eficiência e Risco (Contoso Retail) Este projeto foi desenvolvido para responder a uma dor crítica do setor varejista: O capital parado em estoque que não gera retorno. Através de técnicas avançadas de análise de dados, o objetivo é transformar o banco de dados operacional em uma ferramenta de decisão executiva.
